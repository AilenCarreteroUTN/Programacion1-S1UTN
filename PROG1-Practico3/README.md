## **Práctico n°3:** Arreglos unidimensionales.

El trabajo práctico n°3 se enfoca en la comprensión y manipulación de arreglos unidimensionales en Java, abordando desde su representación en memoria hasta algoritmos de búsqueda y ordenamiento.

---

### 🛠️ **Tecnologías y conceptos cubiertos.**

* **Definición y estructura:** Concepto de arreglo como objeto contenedor de tamaño fijo con elementos del mismo tipo. Gestión de índices y uso de la propiedad `.length`. Diferencia entre declaración e instanciación mediante el operador `new`.

* **Inicialización y tipos:** Manejo de arreglos de tipos primitivos (int, double, boolean) y tipos referencia (objetos/String). Valores por defecto al crear arreglos: 0 para numéricos, false para booleanos y null para objetos.

* **Modularización:** Implementación de programas divididos en métodos (entrada, proceso, salida) para mejorar la legibilidad y reutilización del código.

* **Recursividad:** Recorrido de arreglos mediante bucles para sumatorias, promedios y búsqueda de valores máximos/mínimos. Búsqueda Secuencial, localización de elementos comparando posición por posición.
