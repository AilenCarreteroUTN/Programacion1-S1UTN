## **Práctico n°7C:** Manipulación dinámica de listas.

En esta tercera parte del trabajo práctico n°7 finaliza el módulo de colecciones enfocándose en la versatilidad de `ArrayList<Integer>` para la gestión de datos dinámicos, cálculos estadísticos y la representación de estructuras complejas mediante objetos.

---

### 🛠️ **Tecnologías y conceptos cubiertos.**

* **Gestión dinámica de memoria:** Uso de `ArrayList<Integer>` para almacenar colecciones de tamaño variable. Control de carga de datos mediante centinelas (finalización por ingreso de valores negativos o cadenas específicas como "FIN").

* **Algoritmos estadísticos y de búsqueda:** Cálculo de mayores, menores y rangos en listas dinámicas. Procesamiento de promedios aritméticos y conteo de elementos según su relación con la media. Implementación de la búsqueda secuencial para localizar elementos y sus posiciones por primera vez.

* **Ordenamiento de colecciones:** Investigación y aplicación de métodos para ordenar listas de forma ascendente y descendente.

* **Representación de estructuras complejas:** Modelado de una matriz dispersa mediante una lista de objetos "celda" (fila, columna, valor), permitiendo almacenar solo las posiciones asignadas por el usuario.

---

### 🚀 **Algunos de los desafíos destacados.**

* **Análisis de números aleatorios:** Generación de listas de 20 números pares para realizar análisis estadísticos completos.
* **Sistema de celda y matriz:** Desarrollo de una lógica donde una lista de objetos actúa como una matriz, incluyendo un método de búsqueda por coordenadas que valida si una celda específica ha sido inicializada o no.
