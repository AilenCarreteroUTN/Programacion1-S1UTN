## **Práctico n°2:** Práctica de tipos de datos y funciones de cadena.

El trabajo práctico n°2 profundiza en la manipulación de datos a bajo nivel (casteo y tipos primitivos) y avanza hacia el paradigma de Programación Orientada a Objetos (POO) y la recursividad.

---

### 🛠️ **Tecnologías y conceptos cubiertos.**

* **Casting y tipos de datos:** Conversión de variables double a tipos como short, int, long, float y String. Análisis de desbordamiento de variables fuera de rango. Diferenciación entre datos primitivos (por valor) y tipos de referencia

* **Manipulación de cadenas:** 
  * Uso de métodos esenciales: `length()`, `charAt()`.
  * Comparación avanzada de cadenas mediante `equals()`, `compareTo()`.
  * Extracción y búsqueda de subcadenas con `substring()`, `indexOf()`.
  * Conversión de caracteres a sus respectivos códigos **ASCII**.

* **Programación Orientada a Objetos (POO):** Creación de clases con atributos, métodos get/set y constructores sobrecargados. Implementación de métodos estáticos para la manipulación de fechas (`Date`). Lógica de negocio encapsulada (ej. calculadora aritmética y operaciones con fracciones).

* **Recursividad:** Resolución de problemas mediante llamadas recursivas: suma de números naturales, suma de dígitos de un número e inversión de cadenas de texto.
