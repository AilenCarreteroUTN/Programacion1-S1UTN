## **Práctico n°4:** Arreglos multidimensionales.

El trabajo práctico n°4 aborda la gestión de datos en múltiples dimensiones, permitiendo pasar de la idea de un vector a la de matrices.

---

### 🛠️ **Tecnologías y conceptos cubiertos.**

* **Definición y sintaxis:** Declaración de matrices usando múltiples corchetes: `Tipo[][] nombre = new Tipo[filas][columnas]`. Uso de diversos tipos de datos: int, double, boolean, char y String para conformar las celdas.

* **Manipulación de matrices:** Acceso a elementos mediante índices, recordando que en Java siempre inician en 0. Uso de `.length` para obtener el número de filas y `[0].length` para el número de columnas. Introducción al método .`clone()` para crear copias independientes de una matriz existente.

* **Estructuras N-Dimensionales:** Modelado de datos complejos (como el conteo de coches por hora, día, mes y año) utilizando arreglos de hasta 5 dimensiones.
